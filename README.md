# HR-Analytics-Management-Dashboard
An HR Analytics Management Dashboard built entirely in Excel. This project features interactive data visualizations, KPI tracking, and automated reporting for workforce metrics like attrition, headcount, and performance. Leverages advanced Excel functions, pivot tables, and slicers to transform raw HR data into actionable business insights.
## Project Overview
This project transforms raw HR data into a high-impact, interactive Excel Dashboard. It provides stakeholders with real-time insights into workforce health, helping to identify trends in employee retention, performance, and departmental growth.

## Key Features
Dynamic KPI Tracking: Real-time monitoring of Headcount, Attrition Rate, and Average Salary.

Interactive Visualizations: Slicers and timelines for filtering by Department, Gender, and Tenure.

Performance Analysis: Comparative charts for employee ratings and promotion eligibility.

Data Automation: Built using Power Query for efficient data cleaning and transformation.

## Tools Used
Microsoft Excel: Advanced Formulas (VLOOKUP/XLOOKUP, INDEX-MATCH), Pivot Tables, and Pivot Charts.

Power Query: For ETL (Extract, Transform, Load) processes.

Data Visualization: Custom formatting and interactive slicers for a seamless UI.

## Insights Discovered
Attrition Trends: Identified a 15% higher turnover rate in the Sales department compared to others.

Diversity & Inclusion: Visualized gender distribution across leadership roles.

Recruitment Needs: Highlighted departments with the highest vacancy rates for Q4 planning.

## How to Use
Download the .xlsx file from this repository.

Open the file and ensure "Enable Macros" is selected (if applicable).

Use the Slicers on the left to filter the data and update the visuals instantly.
